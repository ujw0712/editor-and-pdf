# Editor And Pdf
- 에디터로 작성한 내용을 PDF 파일로 다운로드.

# Tech Stack
- Vue3 
- 에디터 (Tiptap)
- pdf 다운로드 (jspdf, html2canvas)
- 공동 편집 (@tiptap/extension-collaboration, yjs, @hocuspocus/provider)

# Script
```
npm install
```
```
npm run serve
```
